## 2.1. 데이터베이스와 테이블 생성
### 2.1.1. 데이터베이스 생성
- DB를 생성하기 위해서는 `CREATE DATABASE` 문을 사용한다.
```SQL
CREATE DATABASE my_database;
```
- "my_database"라는 이름의 DB 생성

### 2.1.2 테이블 생성
- 테이블은 DB내에서 데이터를 구조화하여 하는 용도로 사용된다.
- `CREATE TABLE` 문을 사용하여 테이블을 생성
```sql
CREATE TABLE employees (
	id INT,
	name VARCHAR(50),
	age INT,
	department VARCHAR(50)
);
```
- 테이블 생성시에는 열(Column)의 이름과 해당 열의 데이터 타입을 정의해야 한다.

### 2.1.3 주의사항
1. **데이터베이스 이름 중복 방지**: DB를 생성할 때 다른 DB와 이름이 겹치면 안 된다.
2. **테이블 구조 정확성**: 테이블을 생성할 때 각열(Column)의 이름과 데이터 타입을 정확하게 지정해야 한다.
3. **제약 조건(Constraint) 설정**: NOT NULL 제약 조건, 외래 키(Foreign Key) 제약 조건 등
4. **보안과 권한 설정**
