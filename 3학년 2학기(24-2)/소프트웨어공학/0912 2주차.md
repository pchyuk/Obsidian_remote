## Maintainability (유지 보수성)
- Maintenance
	- Corrective (수정) : 오류 Fixing, 교정
	- Adaptive (적응) : 환경, platform change(HW/SW)
	- Perfective (완전) : code restructring(코드 재구조화)
	- Preventive (예방) : 사고나면 매우 위험한 곳 - 군대, 소방 등등
- 소프트웨어의 진화(유지관리 대신)
	- 유지보수(x) -> 적응/진화

### Evolvability (진화 가능성)

	- 기존 기능을 변경하기 위해
