case Num(n) => k(NumV(n))

case Id(x) => k(env(x))

case Fun(x, b) => k(CloV(x, b, env))

case Add(l, r) =>
	interpCps(l, env, v1 =>
		interpCps(r, env, v2 => {

## State and Reduction
### A state of FAE is a pair of a computation stack and a value stack
- a state is denoted as k||s

### Reduction is a relation over K x S x K x S
- $\rightarrow$ $\subseteq$ K x S x K x S
