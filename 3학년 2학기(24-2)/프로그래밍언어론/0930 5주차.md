### 5.1 Algebraic Data Types (복습)
- 대수적 데이터 타입
- Type은 사실상 무한하기 때문에
- 여러 타입을 묶어서 새로운 걸 만든다.
- Arithmetic Expression(AE)을 묶어서 Algebraic

- AE = Int
- AE + AE = Add
- AE - AE = Sub
	- 책에 오타가 있음 (곱하기 아님)

- type ae = 
	- Num of int
	- Add of ae + ae
	- Sub of ae - ae

- trait / class
