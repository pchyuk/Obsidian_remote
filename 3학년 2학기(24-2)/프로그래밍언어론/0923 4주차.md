## 4.4 First-Class Functions and Lists
##### 리스트의 모든 원소 값 + 1 하는 함수
- def inc1(l: List[Int]): List[Int] = l match {
	- case Nil => Nil
	- case h::t => ==h+1== :: inc1(t)
- }

##### 리스트의 모든 원소 값 제곱 하는 함수
- def square(l: List[int]): List[Int] = l match {
	- case Nil => Nil
	- case h::t => ==h * h== :: square(t)
- } 

- 함수들이 다 비슷비슷하게 생겼다.
