## 1. 두 숫자 리스트의 원소를 각각 더하는 함수를 작성하고, 결과를 출력하라.

```scala
def addLists(xs: List[Int], ys: List[Int]): List[Int] = {
    
}
```

---
### 해답
```scala
def addLists(xs: List[Int], ys: List[Int]): List[Int] = {
    (xs, ys) match {
        case (Nil, _) => ys
        case (_, Nil) => xs
        case (x::xt, y::yt) => (x + y) :: addLists(xt, yt)
    }
}
```

- 해설: 재귀 함수를 사용하여 두 리스트를 순회하면서 각 원소의 합을 리스트로 반환합니다. 
- 예를 들어, addLists(List(1, 2), List(3, 4))는 List(4, 6)을 반환합니다.
